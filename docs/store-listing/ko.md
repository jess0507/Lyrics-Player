## 完整說明（貼到 Full description，≤4000 字元）

```
오디오 가사 플레이어 (Lyrics Player)는 기기 내 파일을 중심으로 한 가벼운 음악 플레이어입니다. 음악 재생에 네트워크도 스트리밍도 필요 없습니다 — 앱을 열고 기기를 스캔하면 바로 재생됩니다. 게다가 가사를 가져오고, 일반 텍스트 가사를 한 줄씩 동기화되는 가사로 자동 변환할 수 있습니다.

🎤 가사 가져오기 및 동기화 표시
• LRC / SRT / VTT / TXT 가사 파일 가져오기
• 동기화 가사는 재생에 맞춰 한 줄씩 강조되고 자동으로 스크롤
• 일반 텍스트 가사도 한눈에 읽기 편하게 표시
• 가사 글자 크기 자유 조절

✨ 가사 자동 동기화 (auto-sync)
• 타임스탬프 없는 텍스트만 있나요? 한 번 탭으로 정렬
• 각 줄을 정확한 재생 시점에 맞춰 스크롤 동기화 가사로 변환
• 직접 입력했거나 웹에서 복사한 가사에 안성맞춤
• 자동 동기화는 로그인이 필요하며, 해당 곡의 압축 오디오를 정렬 서비스로 업로드해 계산합니다(일일 횟수 제한 있음)

🎵 로컬 & 오프라인
• 기기 라이브러리 자동 스캔 또는 파일 수동 가져오기
• 다시 스캔 및 목록에서 제거 지원
• 키워드 검색으로 원하는 곡을 빠르게
• 로컬 음악 재생은 완전히 오프라인으로 동작

🎧 백그라운드 재생
• 다른 앱을 쓰거나 화면을 잠가도 음악이 끊기지 않음
• 알림 재생 컨트롤: 재생/일시정지, 이전/다음 곡

⏩ 정밀한 재생 컨트롤
• 빨리 감기·되감기, 버튼을 누르고 있으면 연속 탐색
• 가변 배속 재생, 한 번 탭으로 원래 속도 복귀
• 동기화 가사와 함께라면 어학 학습, 오디오북, 강의 녹음에 제격

🔁 재생 모드
• 반복과 셔플을 원하는 대로 조합

📊 청취 통계
• 재생 횟수와 누적 청취 시간 기록
• 자주 들은 곡 랭킹, 언제든 초기화 가능

🌍 16개 언어 지원
English, 繁體中文, 简体中文, Español, Français, Deutsch, 日本語, 한국어, Português, Italiano, Русский, Türkçe, हिन्दी, Indonesia, Tiếng Việt, العربية (RTL 레이아웃 포함)

🎨 개인화
• 다크 / 라이트 / 시스템 테마와 사용자 지정 강조 색상

🔐 프라이버시 우선
• 음악 라이브러리는 기기에 보관됩니다. 압축 오디오는 '가사 자동 동기화'를 사용할 때만 타이밍 계산을 위해 업로드됩니다
• 계정은 선택 사항: 게스트로 사용하거나 이메일·전화·Google로 로그인
• 앱 안에서 언제든지 계정과 데이터 삭제 가능

오디오 가사 플레이어 (Lyrics Player)를 내려받아 기기 속 음악을 가장 편안한 청취 경험으로 만드세요.
```
