# 商店文案 — 俄文（ru）

**App 名稱：** Аудиоплеер с текстами

## 簡短說明（貼到 Short description，≤80 字元）

```
Плеер с импортом и авто-синхронизацией текстов, фон, скорость и статистика.
```

## 完整說明（貼到 Full description，≤4000 字元）

```
Lyrics Player — это лёгкий музыкальный плеер для ваших локальных файлов. Чтобы слушать музыку, не нужны ни сеть, ни стриминг: откройте приложение, просканируйте устройство и слушайте. А ещё лучше: импортируйте тексты и автоматически синхронизируйте обычный текст построчно.

🎤 Импорт и синхронный показ текстов
• Импорт файлов LRC / SRT / VTT / TXT
• Синхронные тексты подсвечиваются построчно и прокручиваются сами
• Обычный текст показывается для удобного чтения
• Размер шрифта текста регулируется

✨ Авто-синхронизация текстов
• Есть только текст без отметок времени? Выровняйте его одним касанием
• Привязывает каждую строку к нужному моменту, превращая текст в прокручиваемые синхронные тексты
• Идеально для текстов, набранных вами или скопированных из сети
• Требует входа в аккаунт и загружает сжатую копию аудио трека в сервис выравнивания (с дневным лимитом)

🎵 Локально и офлайн
• Автосканирование библиотеки или импорт файлов вручную
• Повторное сканирование и удаление треков в любой момент
• Быстрый поиск по ключевым словам
• Воспроизведение вашей локальной музыки работает полностью офлайн

🎧 Фоновое воспроизведение
• Продолжает играть, пока вы пользуетесь другими приложениями или блокируете экран
• Управление в уведомлении: воспроизведение/пауза, предыдущий/следующий

⏩ Точное управление воспроизведением
• Перемотка вперёд и назад; удерживайте кнопку для непрерывной перемотки
• Изменяемая скорость со сбросом одним касанием
• С синхронными текстами отлично подходит для изучения языков, аудиокниг и лекций

🔁 Режимы воспроизведения
• Повтор и случайный порядок в любом сочетании

📊 Статистика прослушивания
• Число воспроизведений и общее время прослушивания
• Рейтинг любимых треков, сбрасываемый в любой момент

🌍 16 языков
English, 繁體中文, 简体中文, Español, Français, Deutsch, 日本語, 한국어, Português, Italiano, Русский, Türkçe, हिन्दी, Indonesia, Tiếng Việt, العربية (с RTL-вёрсткой)

🎨 Персонализация
• Тёмная / светлая / системная тема и свой акцентный цвет

🔐 Конфиденциальность прежде всего
• Ваша медиатека остаётся на устройстве; только при авто-синхронизации сжатая копия аудио трека загружается для расчёта тайминга
• Аккаунт необязателен: используйте как гость или войдите по email, телефону или Google
• Удалите аккаунт и данные в любой момент прямо в приложении

Скачайте Lyrics Player и превратите свои локальные музыкальные файлы в самое удобное прослушивание.
```
