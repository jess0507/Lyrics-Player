**App 名稱：** 音声歌詞プレーヤー

## 簡短說明（貼到 Short description，≤80 字元）

```
歌詞をインポートして自動同期、1行ずつスクロール。バックグラウンド再生・倍速・統計。
```

## 完整說明（貼到 Full description，≤4000 字元）

```
音声歌詞プレーヤー (AudioLyrics) は、端末内のファイルを中心とした軽量な音楽プレーヤーです。再生にネットもストリーミングも不要——アプリを開いてスキャンすれば、すぐに再生できます。さらに歌詞をインポートでき、プレーンテキストの歌詞を1行ずつ同期する歌詞へ自動変換します。

🎤 歌詞のインポートと同期表示
• LRC / SRT / VTT / TXT 形式の歌詞ファイルをインポート
• 同期歌詞は再生に合わせて1行ずつハイライト・自動スクロール
• プレーンテキストの歌詞もまとめて読めます
• 歌詞の文字サイズを自由に拡大・縮小

✨ 歌詞の自動同期（auto-sync）
• タイムスタンプのないテキストだけ? ワンタップで整列
• 各行を正しい再生位置に合わせ、スクロール同期歌詞に変換
• 自分で打ち込んだ歌詞やウェブからコピーした歌詞に最適
• 自動同期はログインが必要で、その曲の圧縮音声を整列サービスに送信して計算します（1日の上限あり）

🎵 ローカル＆オフライン
• 端末のライブラリを自動スキャン、手動インポートも可能
• 再スキャンやリストからの削除にも対応
• キーワード検索ですばやく曲を発見
• ローカル音楽の再生は完全オフラインで動作

🎧 バックグラウンド再生
• 他のアプリ使用中や画面ロック中も再生は途切れません
• 通知の再生コントロール：再生/一時停止、前/次の曲

⏩ 精密な再生コントロール
• 早送り・巻き戻し、ボタン長押しで連続シーク
• 倍速再生、ワンタップで原速に復帰
• 同期歌詞と組み合わせれば、語学学習・オーディオブック・講義録音に最適

🔁 再生モード
• リピートとシャッフルを自由に組み合わせ

📊 再生統計
• 再生回数と累計再生時間を記録
• よく聴く曲のランキング、いつでもリセット可能

🌍 16言語対応
English、繁體中文、简体中文、Español、Français、Deutsch、日本語、한국어、Português、Italiano、Русский、Türkçe、हिन्दी、Indonesia、Tiếng Việt、العربية（RTL レイアウト対応）

🎨 パーソナライズ
• ダーク/ライト/システムテーマとアクセントカラー

🔐 プライバシー重視
• 音楽ライブラリは端末内に保持。圧縮音声が送信されるのは「歌詞の自動同期」を使うときのみ（タイミング計算のため）
• アカウントは任意：ゲストのまま、または Email・電話・Google でログイン
• アカウントとデータはアプリ内でいつでも削除可能

音声歌詞プレーヤー (AudioLyrics) をダウンロードして、端末内の音楽を最も快適なリスニング体験に。
```
